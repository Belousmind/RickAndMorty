# 🚀 Rick and Morty Character Search | Test task

### 📸 Демонстрация
🔗 <a href="https://belousmind.github.io/RickAndMorty/index.html" target="_blank">Live Demo</a> <br/>
🔗 <a href="https://github.com/Belousmind/RickAndMorty" target="_blank">Репозиторий</a>

### 📌 Функциональность
✅ Автофокус на поле ввода при загрузке страницы <br/>
✅ Мгновенный поиск — запрос отправляется при вводе более 3 символов <br/>
✅ Поиск по имени (например, "Rick", "Morty") <br/>
✅ Если персонаж не найден (код 404), появляется сообщение "Not found! Try another request." <br/>
✅ Карточки персонажей с информацией (имя, разновидность, статус, дата создания) <br/>
✅ Клики по карточкам ведут на официальную страницу API <br/>
✅ Адаптивность — работает на экранах от 320px до 1920px <br/>
✅ Деплой на GitHub Pages <br/>

### 🛠️ Используемые технологии
⚛ React <br/>
⚡ Vite <br/>
🌐 Fetch API — для запроса данных <br/>
🎨 CSS (Grid, Flexbox, Pseudo-classes) <br/>
📡 Rick and Morty API (https://rickandmortyapi.com/) <br/>
🚀 GitHub Pages — для деплоя <br/>

### 📌 Поддержка браузеров
✔ Поддерживается последняя версия Chrome <br/>
